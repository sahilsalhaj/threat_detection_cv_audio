# yolov8l Evaluation Results

This folder contains the full evaluation results for the yolov8l model, including:

- Annotated images (predictions)
- Metrics summary (YAML)
- Plots (PR, confusion matrix, F1, etc.)
- COCO-format and TXT-format results
- DETAILED_SUMMARY.md (this file)
- results.csv (per-image results)
- args.yaml (evaluation arguments)
- samples/ (selected output images)
