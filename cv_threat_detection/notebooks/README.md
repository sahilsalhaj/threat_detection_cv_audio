# notebooks/

This folder contains Jupyter notebooks for model training, experimentation, and analysis.

- `final_cv_train_models.ipynb`: Notebook for training and evaluating YOLO models on the hornet/bee/wasp dataset. Includes data loading, augmentation, training, and evaluation steps.

**Relation to project:**
Notebooks are used for research, prototyping, and documenting the training process. They are not required for inference but are valuable for improving and retraining models.
